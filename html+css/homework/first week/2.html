  •   HTML5是HTMl5即超文本标记语言或超文本链接标示语言的第五个版本。目前广泛使用的是HTML4.01。

  •   HTMl5草案的前身名为Web Applications 1.0

  •   2004年被WHATWG提出。

  •   2007年被W3C接纳，并成立了新的HTML工作团队

  •   2008年1月22日，第一份正式草案公布。
